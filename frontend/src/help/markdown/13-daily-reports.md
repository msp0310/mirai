# 日報

当日の作業実績を案件・タスクと紐づけて記録し、チーム内で確認する画面です。

## 機能概要

- 初期対象日は当日、初期表示は参照可能なチームメンバー全員です。
- 複数案件の作業時間、まとめ、課題、翌日の予定をMarkdownで記録できます。
- 提出済み日報は内容を確認し、コメントを追加できます。
- 日報明細は案件の作業実績へ連携されます。
- チームカレンダーの非稼働日と個人休暇は提出不要です。実作業がある場合は任意で日報を作成できます。

## 操作方法

- 「自分の日報」または「みんなの日報」の「自分の日報を提出」から作成し、作業明細を1件以上入力して提出します。
- チーム管理者は未提出者へリマインドできます。
- リマインド対象には、非稼働日・個人休暇で提出不要のメンバーを含めません。
- 日報を削除するときは確認画面で対象日と内容を確認します。
- 一覧はAPIでページングされ、表示件数が増えても全件を一度に読み込みません。
