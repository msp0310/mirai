# ヘルプの使い方

このヘルプは、SI案件のスケジュール管理を行うための画面別ガイドです。左メニュー、上部のチーム・案件選択、案件内タブ、管理設定の関係を確認しながら利用してください。

![プロジェクト管理アプリの全体像](/help/screenshots/project-gantt.png)

## 機能概要

- チームごとに案件を管理します。
- 案件ごとにガント、概要、週次報告、課題、作業時間、体制、カレンダー、マイルストーン、履歴を確認できます。
- 個人・プロジェクト・チームの分析と、チーム内の日報を確認できます。
- 管理設定では、チーム、メンバー一覧・ログイン、カレンダー、監査ログを管理します。
- ログインしていない状態ではアプリ本体は表示されません。

## 状態の見方

- 上部の保存状態は、現在の変更が保存済みか未保存かを示します。
- 左メニューの選択状態は、今見ている画面の位置を示します。
- 案件内のタブは、選択中の案件に対するビューです。
- 管理設定はワークスペース全体に対する設定です。

## 操作方法

- 左メニューから案件一覧、案件内ビュー、管理設定、ヘルプを移動します。
- 上部のチーム・案件選択から、作業対象を切り替えます。
- 保存ボタンで現在の変更を保存します。
- 右上のアカウントからログアウトできます。

## 操作ツアー

- 初回ログイン時は、チーム選択から案件を開くまでの基本ツアーを表示します。
- `戻る`、`次へ`で案内を移動し、右上の閉じるボタンまたは`Esc`で終了できます。
- ヘルプ上部の操作ツアーから、基本操作、ガント操作、メンバー向け、PM・PL向け、管理者向けの案内を何度でも再実行できます。
- 表示されるツアーは、ログインユーザーの操作権限に応じて変わります。
