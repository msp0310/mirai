# 入出力

Brabioからの移行、CSV/JSONの書き出し、プロジェクトデータの取り込みを行う機能です。

## 機能概要

- Brabio XLSXを読み込み、現在の案件タスクとして取り込めます。
- タスクCSVを書き出して、外部確認用の一覧にできます。
- プロジェクトJSONを書き出して、環境間の移行や確認に使えます。運用バックアップはサーバー側でDBと添付をまとめて取得します。

## 状態の見方

- Brabio取込は、現在の案件タスクをBrabio内容に置き換える前提です。
- 取込確認では、タスク件数、期間、階層、担当者、追加メンバーを確認します。
- 担当者は必須ではありません。未設定タスクも取り込めます。

## 操作方法

- プロジェクト画面のサイドメニューにある入出力から操作します。
- Brabio XLSXを選択すると確認画面が開きます。
- 内容を確認して取り込みを実行します。
- 取り込み後はガント上にタスクが追加されたことを確認します。
