# ガント

案件のタスク階層、日程、担当、進捗、依存関係を編集するメイン画面です。

![ガント画面](/help/screenshots/gantt-main-tab.png)

## 機能概要

- タスクを階層化して、フェーズ、作業、マイルストーンを管理します。
- 日、週、月単位で表示を切り替えられます。
- タスクバーのドラッグで日程を移動できます。
- 非稼働日を考慮した工数・期間調整に対応します。
- Brabio XLSXを取り込み、移行用のタスク一覧を反映できます。
- 表表示では開始日、終了日、進捗、担当者を列で確認できます。

## 状態の見方

- 左側はタスク名、状態、任意列を表示するタスク表です。
- 右側はタイムラインとタスクバーです。
- 完了タスクはグレー表示されます。
- バーにはタスク名、進捗、担当者名が表示されます。
- 今日位置はタイムライン上で強調されます。
- 初期表示は当日です。表示中の期間にタスクがない場合は、案内から未完了タスクの開始日へ移動できます。

## 操作方法

- タスク追加で新しいタスクを作成します。
- F2またはタスク名クリックでタスク名を編集します。
- 右矢印で階層を下げ、左矢印で階層を上げます。
- 上下キーで選択行を移動します。
- Shift + 上下キーで選択範囲を広げます。
- Deleteで選択タスクを削除します。
- タスクバーのダブルクリックで詳細パネルを開きます。
- 列ボタンから表示列を切り替えます。
- メンバー権限では担当タスクの実績だけを更新し、計画日・階層・担当は変更できません。
