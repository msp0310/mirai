# 案件一覧

チームごとの案件をカード形式で確認し、対象案件へ入るための画面です。

![案件一覧](/help/screenshots/project-portfolio.png)

## 機能概要

- 選択中チームの案件ポートフォリオを表示します。
- 案件の進捗、ステータス、期間、担当体制、次の節目を確認できます。
- 案件を選択すると、その案件のガント画面へ移動します。
- 案件ステータスは計画、進行中、完了済みなどを管理できます。

## 状態の見方

- 管理中プロジェクトは、チーム配下の有効案件数です。
- 平均進捗は、案件内タスクの進捗から算出されます。
- 要対応案件は、完了済みを除き、遅延タスクがあるか進捗が35%未満の案件です。
- お気に入りはよく見る案件の優先表示に使います。

## 操作方法

- チームを切り替えて、チーム配下の案件を確認します。
- 検索で案件名、マイルストーン、遅延タスクを絞り込みます。
- ステータスボタンで案件一覧を絞り込みます。
- 案件カードのGanttボタンから詳細スケジュールへ移動します。
- 要対応欄の「課題で対応」から案件の課題管理を開き、担当者、期限、対応状況を記録します。
